### See-A-Mess

A google spreadsheet based CMS. Designed to operated as an online service only requiring a spreadsheet to specified format published from google drive.

#### Requirements

- [Install node.js](http://nodejs.org/) version `>=0.10.x`
    
#### Install

    $ npm install -g bower
    $ npm install
	$ bower install

#### Debugging the app

    $ npm run dev
    
#### Building the app

    $ npm start
